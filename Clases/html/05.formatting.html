Formato de Texto

Etiquetas para formato de texto:

Negritas

El contenido en negritas sirve para resaltar una parte texto.

En caso de necesitar destacar el contenido, se usa strong: HTML.

Cursivas

Si el texto se quiere poner en cursiva.

Si además queremos enfatizar usamos em: texto enfatizado.

mark

El texto resaltadosale en amarillo por defecto.

Si quiero colocar el texto en naranja debo añadir "style".

small

Para coloca el texto pequeño se usa small.

Texto pequeño solo con p.

Texto S

Texto XS

Texto borrado e insertado

Si colocamos un dato que ya no es correcto o actual podemos tacharlo con texto borrado usando del.

Texto tachado sin del.

El nuevo dato o texto se coloca como insertado con ins:Texto insertado.

Texto insertado sin ins.

Subíndice y superíndice

Subíndice: H2O. Subíndice sin sub: O3.

El área del cuadrado es l2. Superíndice sin sup: x3.
